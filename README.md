# SceneTalk Mandarin

![SceneTalk Mandarin logo](assets/images/scenetalk_mandarin_logo.png)

SceneTalk Mandarin is designed for Mandarin beginners and travelers visiting China, helping you start speaking Chinese in real-life situations. It includes offline content such as scene-based dialogues, key expressions, cultural tips, background articles, and human-recorded Chinese audio, helping you learn common, natural Chinese expressions used in real communication.

The iOS app is available on the App Store in all regions outside mainland China.
